# Make It Real

This repository contains the Make It Real skill.

Use `skills/make-it-real/SKILL.md` when a user has a vague idea and wants help turning it into a clear project plan, delivery scope, requirements, implementation approach, or AI-agent handoff tasks.

Do not preload every reference file. Read the skill first, then load the specific files under `skills/make-it-real/references/` only when the skill calls for them.

When final documents are created, summarize the document path, each document's purpose, and how the user can use those documents with Claude Code, Codex, Cursor, Gemini, or another AI/coding tool.
