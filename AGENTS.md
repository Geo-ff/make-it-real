# Make It Real Agent Instructions

This repository packages the Make It Real skill for AI agents.

Use `skills/make-it-real/SKILL.md` when the user wants to make, build, create, launch, automate, plan, or clarify a project but only has a vague idea or does not know where to start.

Read the skill first, then load only the reference files it names when they are needed:

- `skills/make-it-real/references/question-flow.md`
- `skills/make-it-real/references/document-templates.md`
- `skills/make-it-real/references/ai-coding-task-template.md`

Keep the workflow tool-agnostic. Mention products such as Codex, Cursor, Claude Code, Gemini, or other agents only as examples.

Do not start implementation from a vague idea. Use the skill to clarify the product/project profile, preserve the full vision, define the selected delivery target, summarize for confirmation, and create final documents only after the user confirms.
